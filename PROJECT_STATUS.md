## 💡 技术亮点

1. **前后端分离**: Next.js + FastAPI 现代化架构
2. **类型安全**: TypeScript + Pydantic 双重类型检查
3. **数据库设计**: 规范的关系型数据库设计
4. **API文档**: FastAPI 自动生成 Swagger 文档
5. **响应式设计**: Tailwind CSS 现代化UI
6. **易于部署**: 支持一键启动脚本

## 📝 注意事项

1. **API密钥配置**: 需要配置 `backend/.env` 中的 `ANTHROPIC_API_KEY`
2. **端口占用**: 确保 3000 和 8000 端口未被占用
3. **依赖安装**: 首次运行需要安装依赖，可能需要几分钟
4. **浏览器兼容**: 建议使用 Chrome、Edge、Firefox 最新版本
