# 保研申请助手 - 纯前端演示版

这是一个纯前端演示版本，无需后端，可以直接体验核心功能。

## 功能特性

- ✅ **简历上传**：支持上传简历文件
- ✅ **学校选择**：内置5所知名高校（南京大学、复旦大学、上海交大、清华、北大）
- ✅ **材料生成**：模拟生成个人陈述、推荐信、研究计划书、PPT大纲
- ✅ **实时预览**：生成的材料可以在线预览
- ✅ **一键下载**：支持下载生成的材料

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 访问应用
打开浏览器访问: http://localhost:3000

## 使用说明

1. **上传简历**：点击上传区域，选择你的简历文件
2. **选择学校**：从下拉菜单中选择目标学校和具体项目
3. **选择材料**：勾选需要生成的材料类型
4. **生成材料**：点击"开始生成材料"按钮
5. **预览下载**：在右侧预览生成的内容，可单独下载

## 注意事项

- 这是一个演示版本，使用模拟数据生成内容
- 真实项目需要配置后端API和AI服务
- 生成的材料仅供参考，需要根据实际情况修改
